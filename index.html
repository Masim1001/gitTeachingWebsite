GitHub Tutorial

In this tutorial you will learn from scratch how to use github for hosting your code for version control and collaboration.

1) Firstly download the git. You can may download the git from this link: Download Git



2) Create a folder where you want to do your work and go to that folder's location in CMD or Git Bash.

and now write the command: git init

This command creates an empty Git repository - basically a .git directory with subdirectories for objects , refs/heads , refs/tags , and template files.



3) Create an html file or any type of file of your work.



4) Now write this command in the terminal: git satus (to check the current status of your project)

As you see we need to git add this file to include in what we will be committed.

5) So write the command in the terminal: git add index.html
